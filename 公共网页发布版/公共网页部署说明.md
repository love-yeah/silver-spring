# GitHub Pages 公共网页部署说明

这个文件夹是“银发的春天”网页的 GitHub Pages 发布版。你的 GitHub 用户名是 `cherries`，推荐仓库名使用：

```text
silver-spring
```

发布成功后的公共网址通常是：

```text
https://cherries.github.io/silver-spring/
```

## 文件结构

```text
公共网页发布版
├─ .nojekyll
├─ index.html
└─ media
   ├─ cover.jpg
   └─ main.mp4
```

不要删除或改名 `media` 文件夹，否则封面和视频会加载失败。

## 操作步骤

### 1. 新建 GitHub 仓库

1. 打开 https://github.com/new
2. `Repository name` 填写：`silver-spring`
3. 选择 `Public`
4. 不要勾选 `Add a README file`
5. 点击 `Create repository`

### 2. 上传网页文件

1. 进入刚创建的 `silver-spring` 仓库。
2. 点击 `uploading an existing file`。
3. 把 `公共网页发布版` 文件夹里的这些内容拖进去：
   - `.nojekyll`
   - `index.html`
   - `media` 文件夹
   - `公共网页部署说明.md`
4. 页面底部 `Commit changes` 保持默认即可。
5. 点击绿色按钮 `Commit changes`。

注意：上传后仓库根目录里必须直接看到 `index.html`，不能变成 `公共网页发布版/index.html`。

### 3. 开启 GitHub Pages

1. 进入仓库顶部菜单 `Settings`。
2. 左侧点击 `Pages`。
3. `Source` 选择 `Deploy from a branch`。
4. `Branch` 选择 `main`，右侧目录选择 `/root`。
5. 点击 `Save`。
6. 等待 1 到 3 分钟刷新页面。

看到 `Your site is live at ...` 后，公共网页就发布成功了。

## 说明

- 本版视频已压缩为 720p H.264，约 12.8MB，比原始 158MB 更适合公开网页加载。
- 网页为纯静态页面，不需要服务器后端、不需要数据库。
- 如果后续要替换视频，只要保持文件名为 `media/main.mp4` 即可。
